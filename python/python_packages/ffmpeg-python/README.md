# ffmpeg-python
Python bindings for FFmpeg - with complex filtering support

## documentation
[github](https://github.com/kkroening/ffmpeg-python)
[documentation](https://kkroening.github.io/ffmpeg-python/)

## envrionments
- os : MacOS v12.6
- python: 3.9.12
- ffmpeg-python 0.2.0

## installation
```shell
$ pip install ffmpeg-python
```
